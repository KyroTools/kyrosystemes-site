Politique de confidentialité

Protection et utilisation des données personnelles

1. Introduction

Cette politique explique comment les données personnelles sont collectées et utilisées sur le site Kyro Systèmes.

2. Données collectées

Le site peut collecter les données suivantes :

  • Données de navigation (pages visitées, durée de visite)
  • Données anonymisées via Google Analytics
  • Réponses au formulaire de feedback utilisateur
  • Adresse email si fournie volontairement par l’utilisateur

3. Finalité des données

Les données collectées servent uniquement à :

  • Améliorer le simulateur et les guides proposés
  • Comprendre l’utilisation du site
  • Suivre les performances techniques (analyse d’audience)

Aucune donnée n’est vendue ni utilisée à des fins commerciales.

4. Outils utilisés

Le site utilise les services suivants :

  • Google Analytics (mesure d’audience)
  • Google Forms (formulaire de feedback)
  • GitHub Pages (hébergement du site)

5. Cookies

Google Analytics peut déposer des cookies afin de mesurer l’audience. Vous pouvez désactiver les cookies depuis les paramètres de votre navigateur.

6. Durée de conservation

Les données issues du formulaire de feedback et les emails sont conservés pendant une durée maximale de 12 mois, sauf demande de suppression.

7. Base légale

Le traitement des données repose sur l’intérêt légitime de l’éditeur à améliorer son outil et comprendre l’usage du site.

8. Vos droits

Conformément au RGPD, vous pouvez demander :

  • L’accès à vos données
  • La rectification
  • La suppression

Pour toute demande : mobiworks.fr@gmail.com

9. Hébergement

Le site est hébergé par GitHub Inc. (GitHub Pages), 88 Colin P Kelly Jr St, San Francisco, CA 94107, USA.

Ce site utilise des cookies pour mesurer l’audience (Google Analytics) et améliorer les fonctionnalités.
